Features

Multiple Accounts

Multiple user accounts can be supported.

Types of users:

    Regular user
    Admin

Regular Users

Can log in and log out, Can generate reports of their spending over time, Can create, read, update, and delete (CRUD) expenses they own, and Can not CRUD expenses they do not own.

Admin

In addition to regular users, can read all the saved expenses, including those which they do not own, but can not create, update, or delete expenses they do not own.

Back-end

The back-end provides an interface which is agnostic to any particular front-end client implementation.

Front-end

The front-end is a single-page application which does not refresh the browser after initial load.
